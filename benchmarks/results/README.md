# Benchmark Results

Generated relay load-test artifacts are written here by `benchmarks/relay/scripts/run-relay-load.mjs`.

Typical contents per run:

- `run.json`
- `k6-summary.json`
- `relay-memory.ndjson`
- `memory-summary.json`
- `report.md`

The directory contents are ignored by git so local and CI benchmark runs do not churn the repository history.
