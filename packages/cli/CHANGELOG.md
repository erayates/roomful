# Changelog

## 0.1.0-alpha.1

- Initial alpha release of `@roomful/cli` (EP-22):
  - `roomful init` — scaffold a new Roomful project with package.json, tsconfig, and example code
  - `roomful doctor` — validate Node.js version, @roomful/core installation, relay reachability, and env vars
  - `roomful demo` — point to the public demo app
  - `roomful inspect` — point to Room Inspector in @roomful/devtools
