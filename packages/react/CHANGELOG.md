# @cahoots/react

## 1.0.0

### Major Changes

Initial public release of `@cahoots/react`, the React adapter for Cahoots.

- `CahootsProvider` room context and hooks (`usePresence`, `useSharedState`, `useCursors`, `useAwareness`, `useEvents`).
- Idiomatic React 19 bindings with explicit `react`/`react-dom` peer dependency contracts.
- Workspace dependency on `@cahoots/core`.
