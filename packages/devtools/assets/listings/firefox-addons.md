# Firefox Add-ons Listing

## Summary

Live DevTools diagnostics for Cahoots rooms, peers, state, and event traffic.

## Description

Cahoots DevTools adds a panel to Firefox DevTools for inspecting active Cahoots rooms on the current page.

The panel shows:

- peer presence snapshots
- shared state with diff highlighting
- event logs with sender information and timestamps
- current room connection status and transport type
- a simulated peer toggle for solo testing workflows

## Data collection

This extension does not collect or transmit telemetry or personal data.
