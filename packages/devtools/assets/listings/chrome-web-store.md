# Chrome Web Store Listing

## Short description

Inspect FlockJS rooms, peers, state diffs, and event traffic directly inside Chrome DevTools.

## Full description

FlockJS DevTools adds a dedicated panel to Chrome DevTools for live multiplayer diagnostics.

Use it to:

- inspect connected peers and their presence payloads
- view realtime shared state with diff highlighting
- review event traffic with timestamps and sender metadata
- confirm room status and the active transport layer
- inject a simulated peer for solo testing without a second browser tab

## Privacy

FlockJS DevTools does not collect, transmit, or store user data outside the inspected page and the local DevTools session.
