# create-roomful-app

> Scaffold a new Roomful project with a single command.

Interactive CLI for bootstrapping Roomful projects with one of several starter templates:

- **react-app** — React + Vite
- **vue-app** — Vue 3 + Vite
- **svelte-app** — Svelte + Vite
- **solid-app** — SolidJS + Vite
- **angular-app** — Angular + Vite
- **core-vanilla** — Vanilla TypeScript + Vite

## Usage

```bash
npm create roomful-app@latest
```

Or with a project name:

```bash
npx create-roomful-app my-project
```

Follow the interactive prompts to select a template and install dependencies.
